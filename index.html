LucidDreamer by Kasuko

Android application to help achieve and track lucid dreaming progress.

Lucid Dreamer is an Android application planned to have several features that pertain to the techniques involved with learning and practicing lucid dreaming.
The currently planned features are:

  • Alarm Clock
  • Reality Check Time
  • Dream Notebok
  • Lucidipedia Uploader

Dependencies

Android 1.5

Install

  1. When the first release is viable an .apk will be added to the downloads section.
  2. Download and move the .apk file to your Android device.
  3. Enable installing from a third party.
  4. Install via any app that allows apk installation.
OR install via command line 
pm install com.ionshard.luciddreamer.apk
